This is a simple visualization of an application to the University of Minnesota.

Imagine you are a student who has applied to graduate school. The university you applied to uses an automated algorithm to help make admissions decisions, and has created the following interface to help you understand how the algorithm works. This algorithm works by adding together the applicant’s scores in different areas to create a total score. The algorithm compares this sum to an acceptance threshold it has calculated based on data from previous years. If the applicant’s total score is above this threshold, they are accepted. On the following page you will see the scores for three applicants to the university, along with whether those candidates were admitted. The chart below these scores represent how various factors contributed to the algorithm’s decision.
